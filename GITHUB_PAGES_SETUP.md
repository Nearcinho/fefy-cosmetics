# 🚀 Activar GitHub Pages para Fefy Cosmetics

## Pasos para habilitar el sitio web

### 1. Acceder a la configuración del repositorio
1. Ve a: https://github.com/Nearcinho/fefy-cosmetics
2. Haz clic en la pestaña **"Settings"** (Configuración)

### 2. Navegar a Pages
1. En el menú lateral izquierdo, busca y haz clic en **"Pages"**
2. Está en la sección "Code and automation"

### 3. Configurar el origen (Source)
1. En "Build and deployment", encuentra la sección **"Source"**
2. Selecciona **"Deploy from a branch"**
3. En el desplegable "Branch", selecciona **"master"**
4. En la carpeta, selecciona **"/(root)"**
5. Haz clic en **"Save"**

### 4. Esperar la publicación
- GitHub procesará el sitio (toma 1-5 minutos)
- Recargarás la página y verás un mensaje verde:
  > "Your site is live at https://nearcinho.github.io/fefy-cosmetics"

### 5. Verificar el sitio
- Visita: **https://nearcinho.github.io/fefy-cosmetics**
- ¡Tu sitio estará en línea!

---

## 🔧 Personalizar dominio (opcional)

Si deseas usar un dominio personalizado:
1. En la misma página de Settings > Pages
2. Sección "Custom domain"
3. Ingresa tu dominio: `www.fefycosmetics.com`
4. Haz clic en "Save"

## 🆘 Solución de problemas

**El sitio no carga:**
- Espera 5 minutos y recarga
- Verifica que el branch sea "master"
- Asegúrate de que los archivos estén en la raíz (/)

**Cambios no aparecen:**
- GitHub Pages puede tardar hasta 5 minutos en actualizar
- Prueba hacer "hard refresh" (Ctrl+F5 o Cmd+Shift+R)
